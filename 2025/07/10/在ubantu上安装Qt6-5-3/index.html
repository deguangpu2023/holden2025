在ubantu上安装Qt6.5.3

发表于 2025-07-10

我的ubantu是20.04，首先下载Qt在线安装包
这里采用镜像地址进行下载，避免网络过慢。

镜像地址：
Qt安装包
可以选最新的下载器，我这里下了4.9版本。

安装前准备，把下面的都过一遍，看看有没有安装：
sudo apt-get install build-essential libgl1-mesa-dev
sudo apt install libxcb-xinerama​
sudo apt install libxkbcommon-dev
sudo apt install libgl1-mesa-dev

在ubantu终端里，用以下命令打开，在国内最好设置镜像的下载路径，否则默认国外服务器下载，速度很慢，要好几个小时……

设置镜像的下载路径：

如图所示：

可以选一些必备的组件安装，不然要占空间很大。设置mirror后下载安装飞快。

安装结束打开有一个关于libxcb-cursor0报错，在终端安装这个插件就可以了：

sudo apt-get install libxcb-cursor0

成功打开Qt Creator，祝开发愉快~
